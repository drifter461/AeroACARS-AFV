## 🇩🇪 Deutsch

**v0.11.2 — Pilotenwunsch: Touchdown-Chart zeigt jetzt auch 3 Sekunden NACH dem Aufsetzen.**

### Was ist neu

#### 🛬 Touchdown-Sinkrate-Chart erweitert

Im Landing-Detail-View zeigt das Sinkrate-Chart (Block „TOUCHDOWN") jetzt **3 Sekunden nach dem Aufsetzen** zusätzlich zu den bereits gezeigten ~5 Sekunden davor. Du siehst damit:

- wie die Sinkrate sich direkt nach TD beruhigt (Strut-Compression + Rebound)
- ob das Fahrwerk sauber „gesetzt" hat oder ob ein Hopser folgt
- die volle „Flare-bis-stabil"-Kurve in einem Bild

Datenquelle: `post_touchdown_buffer` (50 Hz, ~10 s Post-TD-Sampling läuft schon seit v0.5.39). Bisher landete das nur in einer lokalen JSONL-Datei für Forensik-Zwecke — jetzt im PIREP-Payload und im sichtbaren Chart.

### Was Piloten tun müssen

**Nichts** — der Auto-Updater holt v0.11.2 automatisch. Settings + Landing-History bleiben erhalten.

Hinweis: die erweiterte Chart-Range erscheint nur für PIREPs die **nach** dem v0.11.2-Update geflogen werden. Bestehende Landungen behalten ihr 5 s-Chart (Pre-TD-only).

Wunsch von ViolonC im Discord — Danke fürs Feedback! 🛬

## 🇬🇧 English

**v0.11.2 — Pilot request: touchdown chart now shows 3 seconds AFTER touchdown too.**

### What's new

#### 🛬 Touchdown sink-rate chart extended

In the landing-detail view the sink-rate chart (block „TOUCHDOWN") now displays **3 seconds after touchdown** in addition to the ~5 seconds already shown before. You can now see:

- how sink rate settles right after TD (strut compression + rebound)
- whether the gear „set" cleanly or a bounce follows
- the full „flare-to-stable" curve in a single image

Data source: `post_touchdown_buffer` (50 Hz, ~10 s post-TD sampling has been running since v0.5.39). Previously this only went to a local JSONL file for forensic purposes — now it's in the PIREP payload and on the visible chart.

### What pilots need to do

**Nothing** — the auto-updater fetches v0.11.2 automatically. Settings + landing history are preserved.

Note: the extended chart range only appears for PIREPs flown **after** the v0.11.2 update. Existing landings keep their 5 s chart (pre-TD only).

Request from ViolonC on Discord — thanks for the feedback! 🛬
